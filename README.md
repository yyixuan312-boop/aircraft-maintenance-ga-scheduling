# Aircraft Maintenance Task Scheduling Using Genetic Algorithm

## 📋 项目简介
本项目针对**1110项飞机维修任务**的调度优化问题，采用遗传算法（Genetic Algorithm, GA）在以下约束下求解最优方案：
- 维修间隔利用率：80%~90%（硬约束）
- 每日人力上限：20工时
目标是最小化日均维修成本，同时降低人力过载风险，提升维修计划的合规性与经济性。

## 📊 数据说明
### 原始数据
- 来源：某公司飞机维修任务数据
- 路径：`data/raw/aircraft_maintenance_raw_tasks.xlsx`
- 核心列：
  1. `Event/Description`：维修任务编号/描述
  2. `EstMH (All)`：任务预估工时（支持`HH:MM`和`0 days HH:MM:SS`格式）
  3. `Interval`：任务最大维修间隔（天）

### 数据清洗步骤
1. 工时格式标准化：将混合格式的预估工时统一转换为小时数
2. 维修间隔修正：空值/无效间隔填充为200天（论文约定的最大间隔）
3. 有效任务筛选：剔除无效数据后，保留前1110项任务（满足作业要求）

## 🧬 遗传算法设计
### 1. 染色体编码
- 每条染色体长度为1110（对应任务总数）
- 每个基因代表任务的维修间隔天数，**直接按硬约束生成在80%-90%利用率区间内**：
  `gap_j ∈ [0.8×max_interval_j, 0.9×max_interval_j]`

### 2. 算法参数
| 参数 | 值 | 说明 |
|------|-----|------|
| 种群大小 | 50 | 标准GA种群规模 |
| 迭代次数 | 100 | 算法收敛所需迭代数 |
| 交叉概率 | 0.8 | 单点交叉概率 |
| 变异概率 | 0.05 | 随机点变异概率 |

### 3. 遗传算子
- **选择**：轮盘赌选择法，基于适应度筛选优质个体
- **交叉**：单点交叉，随机交换染色体片段
- **变异**：随机点变异，变异时仍保证基因满足利用率约束

## 🎯 适应度函数设计（论文原版）
适应度函数为**最大化目标**，公式如下：
`Fitness = - (日均维修成本 + 人力过载惩罚 + 零间隔惩罚)`

### 分项说明
1. **日均维修成本（核心目标）**：
   `sum(MH_j / gap_j)`，其中`MH_j`为任务j的预估工时，`gap_j`为调度间隔，目标是最小化日均工时成本。
2. **人力过载惩罚**：
   `λ × Σ(当日工时 - 20)²`，其中λ=100，对单日工时超过20工时上限的部分施加二次惩罚，避免人力过载。
3. **零间隔惩罚**：
   `C × 违规次数`，其中C=1e6，对间隔为0或负数的非法任务施加固定惩罚，保证调度方案的合法性。
